Development Tools

  • Updated Sep 20, 2015

  • Atom

    A more user-friendly, hackable text editor from GitHub. It's currently free, and leverages many of the same shortcuts found in Sublime.

    Coding Text Editor

  • Sublime Text

    I've used Sublime Text for years--try for free and it's a steal when you're ready to buy it!

    Coding Text Editor

  • Emmet

    Speed up your HTML workflow with the Emmet plugin for Atom and Sublime Text. Check out the demo video for a taste of what you've been missing.

    HTML Productivity

  • Hayaku

    Hayaku for Sublime Text can speed up working with CSS or SASS files by leveraging fuzzy abbreviations for style properties.

    Sublime Text Only CSS/SASS Productivity

  • CodeKit

    CodeKit is a priceless tool that will only set you back ~$30. It saves you serious time by compiling SASS, auto-refreshing, and instantly alerting you to CSS or JavaScript errors.

    Mac Only Coding Productivity

  • Bootstrap

    Using the Bootstrap front-end framework makes it super fast to launch a site/app. Bootstrap covers pretty much everything right out of the box, and its documentation and community makes customizing or troubleshooting easy.

    UI Framework Productivity

  • Bootstrap CDN

    Serve up Bootstrap CSS, JS, and even FontAwesome through this officially recommended CDN.

    Productivity

  • Foundation

    Zurb's front-end framework has some powerful foundational features, which make it worth checking out. However, don't expect quite the ease of implementation or great community support that you'll find with Bootstrap.

    UI Framework Productivity

  • Github

    GitHub is the industry leading collaboration, code review, and code management service for both open source and private projects.

    Git Repo Hosting Collaboration

  • Sublime SFTP

    Coding with Sublime Text? Avoid switching gears by updating files via FTP, FPTS, or SFTP directly from Sublime Text 2 or 3.

    FTP Productivity

  • BrowserStack

    BrowserStack takes a lot of the pain out of performing cross-platform, cross-browser testing.

    Cross-Browser Testing

Research Tools

  • Updated Sep 20, 2015

  • InVision

    My favorite tool for getting feedback on visual designs and prototyping simple interactions and user flows.

    Prototyping Productivity

  • UserTesting

    My favorite tool for remote, unmoderated usability testing. If you need moderated or other types of testing, they offer Enterprise plans with additional features and services.

    Usability Testing

  • UsabilityHub

    Free-to-paid service for getting quick and dirty feedback from people, with the emphasis on dirty.

    Five Second Testing Click Testing Navigation/Workflow Testing Preference Testing

  • Clue

    A fun and easy way to test what people remember after a short first impression with your design.

    Impression Testing

  • Optimizely

    If you don't already have an A/B testing platform that you love, do yourself a favor and check out Optimizely for testing your hypotheses.

    A/B Testing

  • OptimalSort

    A digital card sorting tool from Optimal Workshop, OptimalSort will help you understand how users think your content should be organized.

    Card Sorting

  • Peek

    A service from UserTesting.com, Peek offers a glimpse of real people visiting your website. While less powerful than the paid service, you can request a free peek up to three times a month.

    Easy to design surveys that are fast, fun, and powerful. I've never gotten higher response rates!
