Recommended Tools

UX Design Tools

Treejack

A great Information Architecture (IA) tool from Optimal Workshop, Treejack helps you test and validate your IA structure.

OptimalSort

A digital card sorting tool from Optimal Workshop, OptimalSort will help you figure out how users think your content should be organized.

UserTesting.com

UserTesting.com is, hands-down, my favorite tool for remote, unmoderated usability testing. If you need moderated or other types of testing, they offer Enterprise plans with additional features and services.

Optimizely

If you don't already have an A/B testing platform that you love, do yourself a favor and check out Optimizely for testing the effects of content changes.

Peek

Peek, by UserTesting.com, offers a satisfying glimpse of real people visiting your website. While less powerful than their paid service, you can request a free peek up to three times a month.

Balsamiq

If you're looking for an excellent sketching-to-wireframing tool, look no further--Balsamiq is the way to go.

InVision

InVision is my favorite tool for getting feedback on visual designs and prototyping simple navigation flows.

Analog Design Tools

UX Sketchbook

Grids with browser chrome and room for notes make this a killer sketchbook for your desktop web & app designs.

Phone App Sketchbook

Perfect for sketching out your iPhone or Android phone app.

Tablet App Sketchbook

The perfect sketchbook for rapidly designing your iPad or Android tablet app.

Bic ReAction Mechanical Pencil

(0.5mm) I love this pencil, it's been my go-to sketching tool for the past 5 years.

Pentel EnerGel Pen

With a 0.7mm metal point, this is the best gel pen I've used. If you find a better one, please let me know!

Pentel Hi-Polymer Eraser

The original undo tool.

Digital Design Tools

Keynote

[Mac Only] Keynote is my go-to program for creating quick mock ups and specs.

Adobe Photoshop

Photoshop transforms pixel-pushing from work to play.

Adobe Illustrator

My go-to program for creating scalable, vector graphics.

Sketch

[Mac Only] Sketch is great for creating rapid mock-ups or vector graphics. For my workflow, it fits in somewhere between and complements both Keynote and Photoshop/Illustrator.

Image Optim

[Mac Only] My favorite way to compress images for the Web.

Logitech M305 Wireless Mouse

While I might not recommend this mouse for users with larger hands, the battery life and scrolling experience are two reasons I <3 this mouse.

Developer Tools

Sublime Text

Even though I own Adobe Dreamweaver, Sublime Text has become my editor of choice--try for free and when you're ready to "steal" it, the price is only $70 USD!

Emmet

Speed up your HTML workflow with the Emmet plugin for Sublime Text 2. Check out the demo video for a taste of what you've been missing.

Hayaku

Hayaku for Sublime Text 2 speeds up your ability to work with your CSS or SASS files by leveraging fuzzy abbreviations for style properties.

CodeKit

CodeKit is a priceless tool that will only set you back ~$30. It saves you serious time by compiling SASS, auto-refreshing the page, and instantly alerting you to CSS or JavaScript errors.

Bootstrap

Using the Bootstrap front-end framework makes it super fast to launch a site/app. Bootstrap covers pretty much everything right out of the box, and its documentation and community makes customizing or troubleshooting easy.

Foundation

Zurb's front-end framework has some powerful foundational features, which make it worth checking out. However, don't expect quite the ease of implementation or great community support that you'll find with Bootstrap.

Spectacle

Spectacle makes it super easy to move and resize windows, which is especially helpful when you want to view output and code side-by-side.

Github

GitHub is the industry leading collaboration, code review, and code management service for both open source and private projects.

Sublime SFTP

Coding with Sublime Text? Avoid switching gears by updating files via FTP, FPTS, or SFTP directly from Sublime Text 2 or 3.

Filezilla

If you're not using Sublime, Filezilla is a simple and free FTP solution.

BrowserStack

BrowserStack takes a lot of the pain out of performing cross-platform, cross-browser testing.

Trello

Trello is currently my favorite tool for backlog management. If your agile team is running Kanban style (not to be confused with Gangnam style), check it out.

Online Education

Treehouse

Treehouse provides a fun & valuable way to develop all kinds of technology and design skills.

Codecademy

Codecademy is another great site that will help you learn to code.

Lynda.com

Lynda.com is the place to go when you need design software training.

Coursera

Online courses from top universities & professors.
